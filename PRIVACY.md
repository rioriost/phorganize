# Privacy Policy

Last updated: June 15, 2026

Phorganize is a local macOS app for organizing camera media files. It does not collect, transmit, sell, or share personal data.

## Data collection

Phorganize does not collect personal data.

The app does not use accounts, analytics, advertising SDKs, tracking technologies, telemetry, or third-party cloud services.

## Local file access

Phorganize only accesses folders and files selected by the user, such as a source media folder and a destination archive folder.

The app reads local file metadata, including file timestamps and available media metadata such as capture date, camera model, and lens model. This information is used only to generate destination folders and file names.

Copy and move operations are performed locally on the user's Mac and selected storage devices.

## Data stored on device

Phorganize stores app settings locally, including organization rules and security-scoped bookmarks for previously selected folders. These bookmarks allow the app to restore access to user-selected folders on later launches.

Stored settings remain on the user's device and are not transmitted by Phorganize.

## Network access

Phorganize does not require network access for its app functionality and does not transmit media files, metadata, settings, or usage information.

## Third parties

Phorganize does not share data with third parties.

## User control

Users control which folders Phorganize can access by selecting them in the app or dragging them into the app window.

Users can remove saved app settings through macOS system settings or by deleting the app's stored preferences.

## Contact

For privacy questions, open an issue at:

<https://github.com/rioriost/phorganize/issues>

# プライバシーポリシー

最終更新日: 2026年6月15日

Phorganize は、カメラのメディアファイルを整理するためのローカル macOS アプリです。個人データを収集、送信、販売、共有しません。

## データの収集

Phorganize は個人データを収集しません。

アカウント、分析、広告 SDK、トラッキング技術、テレメトリ、第三者のクラウドサービスは使用していません。

## ローカルファイルへのアクセス

Phorganize は、ユーザーが選択したフォルダとファイルにのみアクセスします。たとえば、分類元のメディアフォルダや分類先のアーカイブフォルダです。

アプリは、ファイルのタイムスタンプ、撮影日時、カメラモデル、レンズモデルなど、利用可能なローカルのメディアメタデータを読み取ります。これらの情報は、分類先フォルダとファイル名を生成するためにのみ使用されます。

コピーおよび移動処理は、ユーザーの Mac と選択されたストレージデバイス上でローカルに実行されます。

## デバイス上に保存されるデータ

Phorganize は、分類ルールや、過去に選択したフォルダの security-scoped bookmark などのアプリ設定をローカルに保存します。これらの bookmark により、次回起動時にユーザーが選択したフォルダへのアクセスを復元できます。

保存された設定はユーザーのデバイス上に残り、Phorganize によって送信されることはありません。

## ネットワークアクセス

Phorganize のアプリ機能にネットワークアクセスは必要ありません。メディアファイル、メタデータ、設定、利用情報を送信しません。

## 第三者提供

Phorganize はデータを第三者と共有しません。

## ユーザーによる制御

ユーザーは、アプリ内でフォルダを選択するか、アプリウインドウへドラッグすることで、Phorganize がアクセスできるフォルダを制御します。

保存されたアプリ設定は、macOS のシステム設定またはアプリの保存済み設定を削除することで削除できます。

## お問い合わせ

プライバシーに関する質問は、以下の GitHub Issues から連絡してください。

<https://github.com/rioriost/phorganize/issues>
